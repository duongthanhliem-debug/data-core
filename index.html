Anthropic API (Claude)
Lưu trong localStorage · console.anthropic.com
📋
Hướng dẫn cài đặt & cập nhật
🔧 Cài đặt lần đầu:
  1. Mở Google Sheet → Extensions → Apps Script
  2. Dán nội dung file Code.gs mới nhất → Lưu
  3. Deploy → New deployment → Web App → Execute as: Me → Who has access: Anyone → Deploy
  4. Copy URL vừa tạo → Dán vào sheet Config cột B dòng "Web App URL"
  5. Vào trang này → nhập URL vào ô trên → bấm Lưu cài đặt → Kiểm tra kết nối
⚠️ Khi deploy GS mới (URL thay đổi):
  1. Cập nhật URL mới vào sheet Config cột B
  2. Bấm nút "🗑 Xóa cache URL" ở trên để xóa URL cũ
  3. Dán URL mới vào ô Web App URL → Lưu cài đặt
  4. Bấm "🧪 Test lưu CalNVL" để xác nhận GS mới hoạt động
